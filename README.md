## üöÄ Getting Started

### Prerequisites
- Node.js 18+
- npm 9+

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/neuron-interface.git

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Configuration

Create a `.env` file:

```env
VITE_ANTHROPIC_API_KEY=your-api-key-here
```
